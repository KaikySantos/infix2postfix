Convert an infix expression to a postfix expression

I made this program for the college <b>data structure</b> discipline.

<br />

Example:
```bash
Input:  7 * 5 + 4
Output: 7 5 * 4 +

Input:  ( 7 + 5 ) * ( 4 / 2 )
Output: 7 5 + 4 2 / *

Input:  ( 7 + 5 ) * 4 + ( 2 - 9 ) / 1 + 8
Output: 7 5 + 4 * 2 9 - 1 / + 8 +
```
